# SpringBoot3 Multi-Module (Java + Kotlin)

> 이 프로젝트는 **Java**와 **Kotlin**을 각각 Spring Boot로 구성하고,  
> 기능을 모듈 단위로 분리하여 붙여서 테스트하기 위한 학습용 멀티모듈 프로젝트입니다.

## 구성

- 루트 프로젝트는 멀티모듈 관리만 담당
- `java-api` : Java 기반 Spring Boot API 모듈
- `kotlin-api` : Kotlin 기반 Spring Boot API 모듈


## 목적
- Java/Kotlin Spring Boot 구조 비교 및 학습
- 기능별 모듈화를 연습하고 조합 실험
- 멀티모듈 빌드/실행 구조 이해
